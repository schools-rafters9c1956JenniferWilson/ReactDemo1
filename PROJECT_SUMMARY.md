## 🎓 课程详细介绍

### 📚 第一课：基础概念

**时间**: 1-2 小时 | **难度**: ⭐

学习内容：

- JSX 语法
- useState 基础
- 条件渲染（if/else、&&）
- 列表渲染（map）
- 表单双向绑定
- 事件处理

**目标**: 能写出简单的 React 组件

---

### 🪝 第二课：Hooks 深入

**时间**: 2-3 小时 | **难度**: ⭐⭐

学习内容：

- useState 进阶（多状态、对象状态）
- useEffect 生命周期
- 依赖数组（关键！）
- useCallback 记忆化
- useMemo 缓存计算
- useReducer 复杂状态
- useContext 全局状态

**目标**: 掌握 React 的核心 Hooks

---

### 🧩 第三课：组件系统

**时间**: 1-2 小时 | **难度**: ⭐⭐

学习内容：

- Props 传递
- 事件处理和回调
- 组件拆分和复用
- 组件通信
- TypeScript 类型定义

**目标**: 能设计可复用的组件

---

### ⚡ 第四课：进阶概念

**时间**: 2-3 小时 | **难度**: ⭐⭐⭐

学习内容：

- Context API 详解
- 性能优化技巧
- 自定义 Hook
- 常见错误及解决
- Vue vs React 完整对比

**目标**: 理解 React 的最佳实践

---

### 📖 快速参考

**时间**: 随时查看 | **难度**: 无

内容：

- 20+ 个 Vue 和 React 的代码对比
- 快速查找实现方式
- 概念映射表

---

### ✅ 实战项目：Todo 应用

**时间**: 4-6 小时 | **难度**: ⭐⭐⭐

包含功能：

- 添加待办事项
- 删除待办事项
- 编辑待办事项
- 标记完成/未完成
- 任务筛选（全部、进行中、已完成）
- 统计信息（总数、完成数、进度）
- 清除已完成
- 清除全部

**技术点**：

- 自定义 Hook（useTodoManager）
- 状态管理
- useCallback 优化
- useMemo 缓存
- 组件化设计
- TypeScript
